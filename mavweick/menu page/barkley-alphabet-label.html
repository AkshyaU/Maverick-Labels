We offer Barkley® alphabetical labels compatible with the most Barkley® color code filing systems. Many color coded filing systems have different color patterns for different product lines.

Be sure you choose the right colors for your system. Compare the color of the label you need to order with the each of the labels available. Check the size, too, as some systems use the same colors in slightly larger and smaller color code label systems. When you find the exact match, simply enter a quantity.
